# ShipGotchi Product Brief

## One-Liner

ShipGotchi is a premium digital toy for builders: paste a GitHub username, hatch a coding companion, and discover its mood, evolution, outfit, and Builder DNA based on how you ship.

## Challenge Fit

The challenge asks for a consumer-facing webapp with at least three working features, no login wall, mobile-friendly execution, and a production URL that judges can test live.

ShipGotchi fits because:

- It is consumer-facing and immediately understandable.
- It uses public GitHub usernames, so there is no login or OAuth risk.
- It naturally supports several interactive features without feeling bloated.
- It is highly visual, which matters for a live judging stream.
- It aligns with Dollar Vibe Club's builder audience and post-sprint launch prize.

## Audience

Primary users:

- Builders, vibe coders, indie hackers, students, and GitHub users.
- DVC stream viewers who want a quick, funny, shareable result.

Judges:

- Need to understand and test the product in under 60 seconds.
- Need to see at least three working features without reading instructions.
- Need a stable mobile-friendly production URL.

## Product Promise

ShipGotchi turns dry public GitHub activity into a living, charming identity moment. It does not ask users to understand analytics. It gives them a creature.

## MVP Scope

The MVP should contain exactly five standout features:

1. GitHub Hatch
2. Pet Mood And Evolution
3. Premium Sprite Moment
4. Builder DNA Card
5. Compare / Party Mode

These features are enough to feel complete, replayable, and demo-worthy without requiring accounts, payments, native apps, extensions, or complex infrastructure.

## What We Are Not Building In The Sprint

- User accounts
- Global leaderboard
- Real desktop overlay pet
- Browser extension
- Native desktop app
- Real-time AI image generation
- GitHub OAuth
- Game economy
- Pet inventory
- Multiplayer persistence

## Long-Term Vision

After the sprint, ShipGotchi can become a persistent builder companion:

- Lives as a web widget, browser extension, desktop app, or IDE sidebar.
- Grows as you ship.
- Celebrates commits, PRs, launches, and streaks.
- Adds friend leaderboards, teams, and seasonal collectible forms.

For the sprint, we only hint at this vision through a web-based "Desk Mode" where the pet can dock in the corner of the page.
